# **App Name**: Course Compass

## Core Features:

- Course Listing: Display available Computer Science courses with titles and descriptions.
- Simple Filtering: Allow users to filter the list of courses by general topic.
- Course Text Analysis: Analyze course titles and descriptions to determine topic clusters and generate tags for the courses.
- AI-Powered Course Suggestion Tool: Suggest closely-related courses to the user based on course content. This will function as a tool by deciding what related course it should show.

## Style Guidelines:

- Primary color: Vibrant purple (#9D4EDD) to convey creativity and intellect.
- Background color: Light purple (#F3E5F5) for a soft and engaging backdrop.
- Accent color: Electric blue (#5978EE) to highlight key elements and interactive components.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and tech-oriented feel.
- Body font: 'Inter' (sans-serif) for readability and clean presentation.
- Use flat design icons related to Computer Science themes such as coding, algorithms, and data structures.
- Design a clean and intuitive layout with clear sections for browsing, filtering, and viewing recommended courses.
